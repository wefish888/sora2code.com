## 🎭 动画效果

### 入场动画
```
Frame 1: scale(0.9) + opacity(0)
Frame 2: scale(0.95) + opacity(0.3)
Frame 3: scale(1.0) + opacity(0.7)
Frame 4: scale(1.0) + opacity(1.0)

时长: 300ms
缓动: ease-out
```

### 脉冲动画（图标）
```
Frame 1: box-shadow: 0 0 20px purple(0.4)
Frame 2: box-shadow: 0 0 40px purple(0.8)
Frame 3: box-shadow: 0 0 20px purple(0.4)

时长: 2s
循环: infinite
```

### 倒计时旋转动画
```
Frame 1: rotate(0deg)
Frame 2: rotate(180deg)
Frame 3: rotate(360deg)

时长: 1s
循环: infinite
```

## 🎬 使用说明

1. 在浏览器访问 `/test-tip-modal` 查看实际效果
2. 点击测试按钮触发弹窗
3. 观察倒计时和动画效果
4. 尝试切换不同币种
5. 测试复制功能
6. 等待10秒后测试跳过按钮

**体验最佳实践**: 在真实场景下测试，感受用户的实际体验！
